# Sistemas BBD - Web One Page

Sitio simple en HTML + CSS + JS con experiencia de scroll continuo:

- Home
- Nosotros
- Valores
- Contacto

Contacto exclusivo por WhatsApp con mensaje precargado.

## Estructura

- index.html
- css/styles.css
- js/main.js

## WhatsApp visible

Actualiza los enlaces en `index.html`:

- `https://wa.me/5490000000000`

Mensaje precargado actual:

- `hola, me interesan sus servicios`

## Prueba local

Abre el archivo en un server estatico y navega a:

- http://localhost:8080

## Deploy en DonWeb

Sube todos los archivos respetando la estructura de carpetas.
